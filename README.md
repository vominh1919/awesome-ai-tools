# Awesome AI Tools 🚀

A curated list of 1000+ AI/ML tools, libraries, and resources.

## Categories

### 🧠 Large Language Models
- [Hugging Face Transformers](https://github.com/huggingface/transformers) - State-of-the-art ML for PyTorch, TensorFlow, and JAX
- [vLLM](https://github.com/vllm-project/vllm) - High-throughput LLM serving
- [Axolotl](https://github.com/axolotl-ai-cloud/axolotl) - Easy fine-tuning of LLMs
- [TRL](https://github.com/huggingface/trl) - Train transformer language models with RL

### 🤖 AI Agents
- [Hermes Agent](https://github.com/NousResearch/hermes-agent) - Intelligent AI assistant
- [LangChain](https://github.com/langchain-ai/langchain) - Build LLM-powered apps
- [AutoGen](https://github.com/microsoft/autogen) - Multi-agent conversation framework
- [CrewAI](https://github.com/joaomdmoura/crewai) - Framework for orchestrating AI agents

### 📊 Training & Evaluation
- [Prime Intellect](https://github.com/PrimeIntellect-ai/verifiers) - Environments for LLM RL training
- [Weights & Biases](https://github.com/wandb/wandb) - ML experiment tracking
- [LM Evaluation Harness](https://github.com/EleutherAI/lm-evaluation-harness) - Unified framework for LLM evaluation

### 🔧 Utilities
- [Unsloth](https://github.com/unslothai/unsloth) - 2-5x faster LLM fine-tuning
- [GGUF](https://github.com/ggerganov/llama.cpp) - Efficient CPU/GPU inference
- [Guidance](https://github.com/guidance-ai/guidance) - Structured generation

### 🧩 AI Applications
- [MindMap AI](https://mindmapai.app/text-summarizer) - Converts documents, PDFs, and text into structured visual mind maps for easier understanding and planning.
